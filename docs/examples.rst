Examples
========

Pre Requisites
---------------

If you want to run the examples below on your own computer, you need to follow the
:ref:`setup` guide. The :ref:`setup` guide covers setting up Globus Compute and GCP on the
machine you are using.


.. note::
   For a quick 5-minute start, you can run the
   `Binder Examples <https://github.com/globus-gladier/gladier-examples>`_,
   which are all run in a pre-built environment and require no setup.
